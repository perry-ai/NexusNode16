### 总体目标

0. **通过项目学习JavaScript等内容**：

1. **完成对 Git 功能的一些辅助功能**：

   - 根据要求创建 tag
   - 比对分支差别
   - 追踪各分支情况

2. **实现一些数据跟踪**：

   - 通过 axios 等方式，对类似电子看板网站上的内容进行爬取
   - 汇总统计成图标展示
   - 追踪如github趋势等前沿咨询

3. **将 Git 代码情况和电子看板内容进行联动分析**：

   - 实现数据的整合和分析

4. **在这些分析过程中，通过 langchainjs，接入远端 LLM 模型进行智能分析**：
   - 集成 langchainjs 和远端 LLM 模型
